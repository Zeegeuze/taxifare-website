## Taxifare

This repo was a project of my bootcamp where we trained ML Ops. We worked with:
* Google Cloud Storage
* Big Query
* Google Computer Engine
* MLFlow
* Uvicorn
* FastAPI
* Google Cloud Run
* Docker
* Streamlit

For this, we:
* We developed an API
* Created a Docker image
* Deployed it on Google
* Created a streamlit frontend
